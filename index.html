Privacy Policy

How iClawd handles your data — the short version: it doesn't collect any.

Effective Date: February 19, 2026

1. Data Collection

  • iClawd does not collect, store, or transmit any personal data to the developer or any third-party analytics service.
  • The app does not include any analytics, tracking, or telemetry SDKs.
  • No user accounts are required. There is no sign-up or login.

2. Local Storage

  • Gateway connection URL and authentication token are stored locally on your device using iOS Keychain or Android Keystore (encrypted, hardware-backed secure storage).
  • An Ed25519 cryptographic key pair is generated and stored locally on your device for gateway authentication. It never leaves the device.
  • User preferences (auto-pronounce toggle, notification toggle) are stored locally on device.
  • No conversation history is persisted — messages exist only in memory during an active session.

3. Network Communications

  • The app connects only to the gateway URL that you configure. This is typically your own self-hosted OpenClaw server (local network, Tailscale, or your own public server). The developer has no access to this connection.
  • If you configure an ElevenLabs API key for premium text-to-speech, the app sends assistant response text to the ElevenLabs API (api.elevenlabs.io) to generate audio. This is optional and uses your own API key. The developer has no access to your ElevenLabs key or the data sent.
  • The app checks for over-the-air updates from Expo's update service (u.expo.dev). This transmits only app runtime version and platform info, no personal data.

4. Microphone & Speech Recognition

  • Microphone access is used solely for voice input during conversations.
  • Speech recognition is processed by Apple's on-device Speech framework (iOS) or the device's built-in speech recognition (Android). Audio is not sent to the developer or any third party beyond what the OS speech framework requires.

5. Camera

  • Camera permission is requested for a planned QR code scanning feature for gateway pairing. It is not currently active. No photos or video are captured or stored.

6. Third-Party Services

  • ElevenLabs (optional, user-configured): Text-to-speech API. Subject to ElevenLabs' privacy policy. Only used if the user explicitly provides an API key.
  • Expo Updates: Over-the-air update delivery. Subject to Expo's privacy policy. Transmits only technical metadata (app version, platform), no personal data.
  • No other third-party services, SDKs, or analytics tools are integrated.

7. Children's Privacy

The app is not directed at children under 13. We do not knowingly collect data from children.

8. Changes to This Policy

Any changes will be posted on this page with an updated effective date.
